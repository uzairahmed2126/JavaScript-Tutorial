# Implementation in JavaScript
---

## We have 2 way to implement JavaScript
1. Inpage JavaScript
```
<html>  
<head>
<script>
document.write("hey");
</script>
</head> 
<body>
<script>
document.write("hello world");
</script>
</body>
</html>
```
---
2. External JavaScript
```
<html>  
<head>
</head> 
<body>
<script src="./js/2.implementation.js"></script>
</body>
</html>
```
